Integrate the plugin to your website and start receiving payments in crypto.

== Description ==

The CityPay Wordpress plugin, allows you to integrate payment page directly into your website.
Your customers will be able to use various crypto currencies to pay for the products and services you offer to them.
The installation of the plugin is easy and intuitive process.

For detailed information and legal text please see https://Citypay.io

== Installation ==

= Prerequisites =

- You must be using WooCommerce 3.0 or newer
- You must enable WordPress pretty permalinks in Settings > Permalinks.

= Installation Instructions =

- Upload the plugin files to WordPress.
- Activate the plugin through the Plugins screen in WordPress.
- Go to WooCommerce -> Settings -> Payments -> CityPay (click Manage/Set up) and fill out required settings.
